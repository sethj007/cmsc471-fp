What is FluLines?
FluLines is an interactive data visualization exploring the phylogeography of H3N2 influenza — the dominant seasonal flu subtype responsible for most severe flu seasons. Phylogeography is the study of how evolutionary history and geographic spread are intertwined: where a virus mutates, and how those mutations travel across the world.

This project was built to make that invisible process visible. Using genomic sequence data and global surveillance records, FluLines lets you watch H3N2 evolve in real time — branch by branch, year by year, country by country.

This visualization was built by Seth Johnston, Aditya Gupta, and Alexander Chen. Alex Chen authored the project proposal and formulated the outline for the visualization. Alongside writing some of the project webpage and touching up various aspects of the visualization. Seth Johnston created the baseline visualization and then focused his work on the phylogenetic tree and the animation. Aditya aided in touching up the visualization such as adding various features to the map to aid in better visualizing the dataset.

This project draws significant influence from the data visualization and genomic tracking capabilities of Nextstrain's seasonal influenza H3n2 analysis, specifically their tracking of the hemagglutinin segment over a twelve-year period. By utilizing their open-source approach to phylodynamics, this project aims to mirror the clarity and real-time scientific communication found in their tracking of viral evolution and transmission patterns.
